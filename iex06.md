## netplanの設定ファイルを修正して 26ビットマスクにする

G8R, G4R, G1R, G2R, G1, G3R, G2, G3 のnetplan設定ファイルを修正

```
sudo nano /etc/netplan/99_config.yaml
```

```
...
      addresses: [192.168.1.X/26]. # <-- /25 を /26 に修正
...
```


## 4班と８班が G4R，G8R のルーティングテーブルを登録

これが成功すると他の班がログインできるようになります。


## ip route コマンドによるルーティングテーブルの確認

```
ip route
```

## ip route コマンドによるルーティングテーブルの追加

```
sudo ip route add 宛先ネットワーク/サブネット via ゲートウェー dev インターフェース名
```

例

```
ip route add 192.168.0.240/28 via 192.168.226 dev enx8857ee65561c
```
