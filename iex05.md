## 各班担当のルータとホストのIPアドレスを修正する

例　1班は，G1,　G1R　を担当
　　２班は，G2,　G2R　を担当
　　...
　　10班は，G10，　G10R　を担当


### netplan の設定ファイルを修正

24ビットマスクを25ビットマスクに修正する

```
sudo nano /etc/netplan/99_config.yaml
```

例

```
network:
  version: 2
  renderer: networkd
  ethernets:
    eth0:
      dhcp4: false
      dhcp6: false
      addresses: [192.168.1.4/25]
      gateway4: 192.168.1.1
      nameservers:
        addresses: [8.8.8.8, 8.8.4.4]
    eth1:
      dhcp4: false
      dhcp6: false
      addresses: [192.168.1.17/25]
      nameservers:
        addresses: [8.8.8.8, 8.8.4.4]
```

### netplan 実行

```
sudo netplan apply
```
